# AGENTS.md — UAP Disclosure Archive

## Project Purpose

This is a **live-updating local website** that automatically tracks, downloads, and AI-analyzes U.S. government UAP/UFO document releases from [war.gov/medialink/ufo](https://www.war.gov/medialink/ufo/).

On **May 8, 2026**, the U.S. government released **118+ classified documents** in a single mass declassification covering the FBI, Department of War, NASA, and Department of State — spanning incidents from 1942 to 2024. This project downloads them, splits them into 20-page chunks, analyzes every page with Claude AI, and presents summaries on a polished alien/apocalyptic-styled website.

## Architecture

```
/
├── server.js              # Express server (port 5001), serves /public and all /api routes
├── process_pdfs.cjs       # Claude AI analysis — processes all unanalyzed PDFs
├── watcher.cjs            # Polls war.gov hourly, auto-downloads new PDFs
├── download_war_ufo_pdfs.cjs  # Downloads PDFs listed in manifest
├── split_war_ufo_pdfs.py  # Splits each PDF into 20-page chunks
│
├── public/
│   ├── index.html         # Single-page site with nav, hero, ticker, sections
│   ├── style.css          # Alien/apocalyptic theme: black bg, green terminal text, glitch, scanlines
│   └── app.js             # Vanilla JS: fetches API, renders docs/timeline/quotes, modal
│
├── data/
│   ├── timeline.json      # 34 pre-populated historical UAP events (1942–2026)
│   ├── quotes.json        # 15 pre-populated quotes from politicians/whistleblowers/media
│   └── summaries/         # Per-document AI analysis JSON (generated by process_pdfs.cjs)
│
└── war_ufo_pdf_manifest.json  # Master list of 119 documents with agency, URL, release date
```

## API Routes (server.js)

| Endpoint | Returns |
|----------|---------|
| `GET /api/stats` | Total docs, processed count, page count, agency breakdown |
| `GET /api/manifest` | All 118 documents with status (pending/processing/processed), display titles, chunk counts |
| `GET /api/document/:id` | Full AI analysis JSON for a single document |
| `GET /api/timeline` | 34 timeline events |
| `GET /api/quotes` | 15 quotes |
| `GET /api/events` | SSE stream — emits `document_processed` and `new_documents` events |

## Design System

The site uses an **alien/apocalyptic** visual theme:
- Pure black background with CSS scanline overlay
- Matrix green (`#00ff41`) as primary accent with glow effects
- Alert red (`#ff0033`) for warnings and classified stamps
- Amber (`#ffb300`) for secondary highlights
- `Space Mono` monospace font throughout
- Glitch animation on hero title
- Per-agency color coding: FBI=purple, DOW=green, NASA=blue, State=amber

## Key Decisions

- **No framework** — vanilla JS/HTML/CSS only for maximum simplicity and portability
- **Resumable processing** — `process_pdfs.cjs` saves progress after each chunk so interruption is safe
- **SSE live updates** — processed documents appear on the website in real-time without page refresh
- **Port 5001** — 3000 and 5000 were occupied on the host machine; configurable via `PORT` env var
- **0.0.0.0 binding** — server accessible from phone/other devices on same WiFi at `http://192.168.1.98:5001`
- **PDFs excluded from git** — PDFs total ~2.2GB; run `node download_war_ufo_pdfs.cjs` then `python3 split_war_ufo_pdfs.py` to restore
